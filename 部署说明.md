# 部署说明

## 默认后台入口

默认后台地址：

```text
/manage-8f3k2
```

默认账号：

```text
admin
```

默认密码：

```text
ChangeMe_2026_admin
```

上线后建议立刻修改 `.env`：

```env
ADMIN_PATH="/your-private-admin-path"
ADMIN_USER="your_admin_name"
ADMIN_PASSWORD="your_long_random_password"
```

注意：

```text
/admin 会返回 404
robots.txt 不会暴露后台地址
前台页面不会显示后台入口
```

## 启动

```bash
docker compose up -d
```

首次部署后先打开主页：

```text
https://你的域名/
```

如果还没有配置数据库，系统会自动进入初始化页面：

```text
/install
```

填写 MySQL 连接信息后，系统会自动创建数据库表并保存到：

```text
data/install.json
```

默认安装口令：

```text
ChangeMe_Install_2026
```

使用默认安装口令时，安装页面可以不填“安装口令”。如果你把 `.env` 里的 `INSTALL_TOKEN` 改成自定义值，安装时才需要填写。

初始化成功后再打开后台：

```text
https://你的域名/manage-8f3k2
```

如果你不想用安装向导，也可以手动导入：

```bash
mysql -u your_user -p < sql/init.sql
```

## 自动热点导入

worker 默认会运行热点导入：

```text
Google Trends
→ GDELT 相关新闻发现
→ 自动生成 SEO 新闻页面
```

手动执行：

```bash
npm run import:hot-news
```
